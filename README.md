## 💡 Key Insights

- **Consumer segment** accounts for nearly half of all sales (48%)
- **COD** is the most used payment method (43%), indicating a preference for pay-on-delivery
- **Standard Class** shipping dominates — customers prioritize cost over speed
- **Office Supplies** is the top revenue category at 0.64M
- Sales and profit **peak in Nov–Dec**, driven by seasonal/holiday demand
- **West region** leads all regions with a 33% sales share

## 🚀 Getting Started

```bash
git clone https://github.com/adityaaapratappp/Super-Sales-Dashboard-.git
```

1. Open `Super Store Sales Dashboard.pbix` in Power BI Desktop
2. Update the data source path to `SuperStore_Sales_Dataset.csv` if prompted
3. Click **Refresh** to load data
